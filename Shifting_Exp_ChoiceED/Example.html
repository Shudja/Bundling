This example displays two sets of four different urns. Each urn was created before the start of the experiment according to their creation rules. For Urn A in the first set, the computer flipped a computerized coin that had an equal chance of landing on heads or tails. If the coin landed on heads, then Urn A in the first set has 89 green balls and 11 purple balls inside of it. If the coin landed on tails, then Urn A in the first set has 11 green balls and 89 purple balls inside of it. For Urn B in the first set, the computer flipped another, separate, computerized coin that had an equal chance of landing on heads or tails. If the coin landed on heads, then the Urn B in the first set has 63 green balls and 37 purple balls inside of it. If the coin landed on tails, then Urn B in the first set has 37 green balls and 63 purple balls inside of it. Urn D in the first set is also created using a computerized coin. Notice that Urn C in the first set has a 100% chance of having 45 green balls, so you know it has 45 green balls and 55 purple balls. The urns in the second set each have a 100% chance of having their displayed composition.

After reading each urn's creation rule, you would then make two decisions regarding each set in this example Situation. You will first decide which urn to draw from in each set. After making this decision, you will then be shown the color of the ball drawn from each chosen urn and you will be paid for any green balls that you draw. These balls will then be placed back in their appropriate urns. You would then again decide which urn to draw from in each set.
